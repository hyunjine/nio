## 목적
기능 목적 : 내 일상 속 주로 사용하는 앱 기능을 한 앱으로 관리<br>
프로젝트 목적 : 최신 기술을 접목하고 IDE, kotlin, 라이브러리 버전을 최신으로 다룸으로 인해 신기능이나 버그를 체험하는 공간
